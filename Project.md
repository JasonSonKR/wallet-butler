# Project Overview: Wallet Butler (Lovely Ledger)

단순한 기록을 넘어, 삶을 통제하고 자산을 키우는 **금융 비서(Butler)** 서비스입니다.
귀엽고 직관적인 디자인(Lovely Ledger)으로 자산 관리의 즐거움을 제공합니다.

## 🌟 제품 철학 (Product Philosophy)
기존 가계부와 차별화된 5가지 핵심 가치를 지향합니다:
1. **예산 vs 지출 비교**: 단순 기록이 아닌, 설정한 목표(예산) 대비 실제 지출 현황을 비교하고 통제합니다.
2. **비용 구조 분석**: 비용의 총량뿐만 아니라 카테고리별 비중을 분석하여 소비 패턴을 파악합니다.
3. **돌발 비용(Impulse Cost) 관리**: 예산에 포함하기 어려운 충동/돌발 지출을 별도로 격리하여 관리합니다. ("지름신" 관리)
4. **이벤트 비용(Event Cost) 관리**: 매년 통상적으로 발생하는 예측 가능한 경조사/이벤트 비용을 별도 예산으로 관리합니다.
5. **자산 성장(Asset Growth)**: 수입 대비 지출 관리와 금융/부동산 자산 현황을 통해 순자산 성장을 시각화합니다.

## 🚀 현재 구현된 기능 (Current Status)
### 1. 📅 가계부 (Calendar)
- **기록과 흐름**: 월별 달력을 통해 일자별 수입/지출/예산 흐름 파악.
- **스마트 태깅**: 공휴일 자동 표시, 지출/수입/예산 상태를 색상 점(Dot)으로 시각화.
- **거래 관리**: 반복 거래(매주/매월) 자동 생성, 예산의 지출 전환 기능.

### 2. 🎯 예산 (Budget)
- **통제와 계획**: 월별 생활 예산 관리 및 잔액 확인.
- **이벤트 예산 분리**: 매월 반복되는 생활비와 연간 비정기 지출(경조사 등)을 분리하여 관리.
- **주차별 분석**: 해당 월의 주차(4~5주)를 자동 계산하여 주별 예산/지출/예상 금액 분석.
- **돌발 비용 격리**: "지름신이 다녀간 흔적" 섹션을 통해 예산 외 지출을 별도로 집계.

### 3. 📊 통계 (Analysis)
- **지출 성격 분석**: 생활비, 이벤트, 투자, 돌발 비용 등 지출의 성격(Nature)별 비중을 파이 차트로 시각화.
- **카테고리 랭킹**: 지출이 많은 카테고리 순위 확인.
- **캐릭터 피드백**: 지출 패턴(돌발 지출 비율)에 따라 표정이 변하는 '낭비 캐릭터' 위젯.

### 4. 💰 자산 (Asset)
- **포트폴리오**: 현금, 은행, 부동산, 주식 등 자산 항목 관리 및 총자산 계산.
- **관리 기능**: 자산 추가 시 색상 지정 가능, 최대 10개 제한.
- **자동 집계**: 수입/지출 내역을 실시간으로 반영하여 현재 시점의 순자산 자동 계산.

### 5. ⚙️ 설정 (Settings)
- **데이터 관리**: JSON 파일로 데이터 백업(내보내기) 및 복원(가져오기) 기능.
- **초기화**: 데이터 전체 초기화 기능.

- **직관성**: 복잡한 회계 용어 대신 쉬운 용어와 UI 사용.
- **즐거움**: 귀여운 디자인과 인터랙션으로 가계부 쓰는 재미 부여.
- **유연성**: 반복 거래와 예산 계획 기능을 통해 체계적인 관리 지원.
